1. Para datos históricos
   - International Football Results from 1872 to 2023: https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017, excelente para obtener historial de enfrentamientos directos, goles, localía.
   - FIFA World Ranking 1993-2023: https://www.kaggle.com/datasets/cashncarry/fifaworldranking, Evolución del ranking FIFA.
   - Partidos Históricos de Mundiales (1974-2022): https://www.kaggle.com/datasets/ibrahimshahrukh/fifa-world-cup-matches-19742022-dataset, FIFA World Cup Matches (1974–2022) on Kaggle. Ideal para modelos que predicen resultados de partidos individuales.
   - Resultados de Partidos Internacionales (1872-2024): https://github.com/martj42/international_results, GitHub - martj42/international_results. Un dataset masivo con más de 49,000 partidos internacionales, perfecto para modelos más complejos.
   - Datos de la Fase de Grupos y Probabilidades Base (WC 2026): https://www.kaggle.com/datasets/sarazahran1/wc2026-match-probability-baseline-dataset, WC2026 Match Probability Baseline Dataset on Kaggle. Este dataset ofrece un punto de partida con predicciones de probabilidad para cada partido de la fase de grupos basadas en el sistema de Elo Rating.
   - Datos de la Copa del Mundo y Clasificatorias: https://github.com/openfootball/worldcup/#start-of-content, GitHub - openfootball/worldcup. Este repositorio contiene datos de todos los mundiales, incluyendo la edición de 2026, en un formato de texto estructurado.
   - StatsBomb Open Data: https://github.com/statsbomb/open-data, Dataset gratuito con xG, presiones, progresiones de pase y métricas tácticas avanzadas para ~4,000 partidos internacionales. Accesible mediante la librería `statsbombpy`.
2. APIs (Para datos actualizados y detallados):
   - API-Football (RapidAPI): https://rapidapi.com/collection/football-soccer apis (Pueden sacar estadísticas de jugadores en ligas europeas, xG (Expected Goals), tiros al arco, etc. Requiere cuenta gratuita con límite de peticiones).
   - Football-Data.org: https://www.football-data.org/ (Muy buena para
     estadísticas de ligas domésticas donde juegan los convocados).
3. Web Scraping Avanzado (Opcional):
   - FBref: https://fbref.com/en/ (Para métricas avanzadas de jugadores: xA, progresiones de pase, pressings).
   - Transfermarkt: Para obtener el "Valor de mercado" de las plantillas.
   - EloRatings.net: Muchos expertos prefieren el sistema Elo sobre el de FIFA por ser más dinámico y preciso.
